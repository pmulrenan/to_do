📝 Modern To-Do App
A sleek, modern 2026-style To-Do application built with pure HTML, CSS, and JavaScript.
This project focuses on:
Clean UI design
Glassmorphism styling
Smooth micro-interactions
Lightweight, framework-free implementation
🚀 Live Demo
The app is deployed using Vercel:
👉 https://to-do-two-flax.vercel.app
Any updates pushed to the connected GitHub repository automatically trigger a new deployment on Vercel.
✨ Features
✅ Add tasks (press Enter or click Add)
✔️ Mark tasks as complete
❌ Delete tasks
🧹 Clear all completed tasks
📊 Live task counter (remaining + total)
🎨 Modern gradient background + glass UI
⚡ Fast and lightweight (no frameworks)
🛠 Tech Stack
HTML5
Modern CSS (Glassmorphism + Gradients + Animations)
Vanilla JavaScript
Vercel for deployment
No frameworks. No build tools. No dependencies.
📦 Deployment
This project is deployed with Vercel and connected to GitHub for automatic deployments.
Deployment Flow:
Push changes to the connected GitHub repository
Vercel automatically builds and deploys
Updated site goes live at:
https://to-do-two-flax.vercel.app
